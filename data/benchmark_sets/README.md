# Confounder benchmark sets

JSON subsets for qualitative evaluation of ambiguous functional-group pairs.

Generated by:

```powershell
python -m ml.external build-confounder-benchmarks --sqlite-index data/experimental/merged_external_index.sqlite
```

Each `*.json` file lists representative `reference_id` values, expected qualitative behavior, and known ambiguities. Use with report batch mode and manual review before promoting external data to training.

## Targeted manifests (negative-space design)

| Manifest | Problem |
|----------|---------|
| `nitro_vs_noxide_manifest.json` | nitro vs N-oxide / nitroso / heteroaromatic |
| `amide_vs_enamine_manifest.json` | amide vs enamine / pyrrole / imide |
| `siloxane_vs_CO_manifest.json` | siloxane vs ether-ester / C–O polymers |

Refresh coverage counts:

```powershell
python -m ml.external summarize-confounder-coverage
```

See `docs/TARGET_EXTERNAL_EXPANSION.md` for minimum counts and example compounds.
