# Code of Conduct

We are committed to a respectful, inclusive collaboration environment focused on
rigorous spectroscopy and reproducible science.

## Expected behavior

- Critique ideas and evidence, not people.
- Document assumptions when interpreting ambiguous FTIR regions.
- Credit external datasets and prior work in methods text.

## Unacceptable behavior

Harassment, discrimination, sharing proprietary data without permission, or
bypassing access controls on the private repository.

## Enforcement

Maintainers may remove access for violations. Report concerns to repository owners.
